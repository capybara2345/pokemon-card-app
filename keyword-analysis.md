# 포켓몬 포켓 카드 키워드 분류 분석

> 구글 시트의 `기술추가효과` / `특성효과` 전수 분석 결과  
> 이 분류를 기준으로 구글 시트 `키워드` 컬럼에 입력 권장

---

## 키워드 입력 규칙

- 해당 카드에 맞는 키워드를 **쉼표(,)로 구분**하여 입력
- 복수 키워드 가능 (예: `독 부여,벤치 피해`)
- 기술추가효과와 특성효과 **모두 고려**하여 키워드 부여

---

## 제안 키워드 목록

### 🔴 상태이상 부여

| 키워드 | 설명 | 효과 패턴 예시 |
|--------|------|----------------|
| `독 부여` | 상대를 독으로 만듦 | "상대의 배틀 포켓몬을 독으로 만든다" |
| `화상 부여` | 상대를 화상으로 만듦 | "상대의 배틀 포켓몬을 화상으로 만든다" |
| `마비 부여` | 상대를 마비로 만듦 | "상대의 배틀 포켓몬을 마비로 만든다" |
| `혼란 부여` | 상대를 혼란으로 만듦 | "상대의 배틀 포켓몬을 혼란으로 만든다" |
| `잠듦 부여` | 상대를 잠듦으로 만듦 | "상대의 배틀 포켓몬을 잠듦으로 만든다" |
| `자신 잠듦` | 사용 후 자신이 잠듦이 됨 | "이 포켓몬을 잠듦으로 만든다" |
| `자신 혼란` | 사용 후 자신이 혼란이 됨 | "이 포켓몬을 혼란으로 만든다" |

### 🛡️ 상태이상 면역 / 데미지 감소

| 키워드 | 설명 | 효과 패턴 예시 |
|--------|------|----------------|
| `특수상태 면역` | 특수 상태가 되지 않음 | "이 포켓몬은 특수 상태가 되지 않는다" / "에너지가 붙어 있는 포켓몬 전원은 특수 상태가 되지 않는다" |
| `잠듦 면역` | 잠듦이 되지 않음 | "이 포켓몬은 잠듦이 되지 않는다" |
| `기술 데미지 면역` | 특정 기술 데미지를 받지 않음 | "기술의 데미지나 효과를 받지 않는다" / "벤치에 있는 한 기술의 데미지를 받지 않는다" |
| `데미지 감소` | 받는 데미지를 줄임 | "받는 기술의 데미지를 -10/-20/-30/-40/-50/-100한다" |
| `HP 만땅 데미지 감소` | HP가 가득한 상태일 때 데미지 감소 | "HP가 가득한 상태라면 데미지를 -40한다" |
| `기술 효과 면역` | 기술의 효과를 받지 않음 | "상대의 포켓몬이 사용하는 기술의 효과를 받지 않는다" |
| `첫 피격 면역` | 첫 번째 기술 데미지를 받지 않음 | "처음으로 기술의 데미지를 받을 때 그 데미지를 받지 않는다" |
| `동전 피격 방어` | 동전으로 데미지 무효/감소 | "동전을 1번 던져서 앞면이 나오면 데미지를 받지 않는다" |

### ⚔️ 조건부 추가 데미지

| 키워드 | 설명 | 효과 패턴 예시 |
|--------|------|----------------|
| `에너지 기반 피해` | 에너지 개수에 비례한 추가 데미지 | "에너지의 개수 × N데미지를 추가" / "강철/물/풀/번개에너지의 개수 × N데미지" |
| `HP 기반 피해` | 남은 HP 조건에 따른 추가 데미지 | "남은 HP가 N 이하라면 N데미지를 추가" / "데미지를 받고 있다면 N데미지를 추가" |
| `벤치 수 기반 피해` | 벤치 포켓몬 수에 비례한 추가 데미지 | "벤치 포켓몬의 수 × N데미지" / "진화 포켓몬의 수 × N데미지" |
| `특정 포켓몬 기반 피해` | 벤치에 특정 포켓몬이 있을 때 추가 데미지 | "벤치에 「○○」가 있다면 N데미지를 추가" |
| `트래쉬 기반 피해` | 트래쉬 카드 수에 비례한 추가 데미지 | "트래쉬에 있는 서포트의 장수 × N데미지" / "트래쉬에 있는 초포켓몬의 장수 × N데미지" |
| `포인트 기반 피해` | 획득 포인트 수에 비례한 추가 데미지 | "가져간 포인트의 수 × N데미지를 추가" |
| `패 수 기반 피해` | 패 장수 조건에 따른 추가 데미지 | "패가 홀수/짝수 장수라면 N데미지를 추가" |
| `필드 에너지 기반 피해` | 필드 전체 에너지 조건부 추가 데미지 | "필드에 번개에너지가 4개 이상 있다면 N데미지를 추가" |
| `스타디움 기반 피해` | 스타디움 조건부 추가 데미지 | "스타디움이 나와 있다면 N데미지를 추가" |
| `동전 기반 피해` | 동전 결과에 따른 데미지 | "동전을 N번 던져서 앞면이 나온 수 × N데미지" |
| `데미지 반사` | 받은 데미지를 상대에게 돌려줌 | "받고 있는 데미지만큼 상대의 배틀 포켓몬에게 데미지를 준다" |
| `자신 데미지 비례 피해` | 자신이 받은 데미지에 비례한 추가 | "이 포켓몬이 받고 있는 데미지만큼 데미지를 추가" |
| `이전기술 시너지 피해` | 이전 차례에 특정 기술 사용 시 추가 | "이전 차례에 「스위트릴레이」를 사용했다면 N데미지를 추가" |
| `충전 피해` | 다음 차례에 데미지가 증가 | "다음 차례에 「○○」의 데미지를 +N한다 (중복됨)" |

### 💥 벤치 피해 / 자해

| 키워드 | 설명 | 효과 패턴 예시 |
|--------|------|----------------|
| `벤치 피해` | 자신의 벤치 포켓몬에게 데미지 | "자신의 벤치 포켓몬 1마리에게도 N데미지를 준다" / "벤치 포켓몬 전원에게도 N데미지" |
| `상대 벤치 피해` | 상대의 벤치 포켓몬에게 데미지 | "상대의 벤치 포켓몬 전원에게도 N데미지를 준다" |
| `자해` | 자기 자신에게 데미지 | "이 포켓몬에게도 N데미지를 준다" |
| `포인트 차단` | 기절 시 상대 포인트 획득 방지 | "앞면이 나오면 상대는 포인트를 가져갈 수 없다" |
| `기사회생` | 기절 직전 HP를 남기고 생존 | "앞면이 나오면 기절하지 않고 남은 HP가 10인 상태로 남는다" |
| `반격 피해` | 기절 시 / 피격 시 상대에게 데미지 | "기술의 데미지를 받아 기절했을 때 기술을 사용한 포켓몬에게 N데미지" |

### 💚 HP 회복

| 키워드 | 설명 | 효과 패턴 예시 |
|--------|------|----------------|
| `자신 회복` | 자신 또는 아군 포켓몬 HP 회복 | "이 포켓몬의 HP를 N회복" / "자신의 포켓몬 전원의 HP를 N회복" |
| `상태이상 회복` | HP 회복 + 특수 상태 회복 | "HP를 N회복하고 특수 상태도 모두 회복" |
| `HP 회복 금지` | HP가 회복되지 않음 | "서로의 포켓몬 전원의 HP는 회복되지 않는다" |

### ⚡ 에너지 관련

| 키워드 | 설명 | 효과 패턴 예시 |
|--------|------|----------------|
| `에너지 부착` | 에너지존에서 포켓몬에게 에너지를 붙임 | "에너지존에서 N에너지를 N개 내보내 포켓몬에게 붙인다" |
| `에너지 이동` | 에너지를 다른 포켓몬에게 옮김 | "벤치 포켓몬에서 에너지를 1개 선택해서 배틀 포켓몬에게 바꿔 붙인다" |
| `에너지 트래쉬` | 자신의 에너지를 트래쉬함 | "이 포켓몬에서 에너지를 모두 트래쉬한다" |
| `에너지 회수` | 트래쉬에서 에너지를 가져옴 | "트래쉬에서 N에너지를 N개 붙인다" |
| `후퇴비용 감소` | 후퇴 에너지를 줄이거나 없앰 | "후퇴에 필요한 에너지를 1개 적게 / 모두 없앤다" |
| `후퇴비용 증가` | 상대의 후퇴 에너지를 늘림 | "상대 배틀 포켓몬의 후퇴에 필요한 에너지를 1개 많게 한다" |
| `에너지 부착 페널티` | 에너지 부착 시 상대에게 페널티 | "에너지존에서 포켓몬에게 에너지를 붙일 때마다 그 포켓몬에게 N데미지" |
| `에너지 2배` | 에너지 1개를 2개 분으로 취급 | "풀에너지는 각각 풀에너지 2개분으로 작용한다" |
| `기술비용 감소` | 기술 사용 에너지 비용 감소 | "기술을 사용하기 위한 에너지를 풀에너지 1개만큼 적게 만든다" |

### 🔄 교체

| 키워드 | 설명 | 효과 패턴 예시 |
|--------|------|----------------|
| `자신 교체` | 자신의 배틀 포켓몬을 벤치와 교체 | "이 포켓몬을 벤치 포켓몬과 교체한다" |
| `상대 강제 교체` | 상대의 배틀 포켓몬을 강제로 교체 | "상대의 배틀 포켓몬을 벤치 포켓몬과 교체한다" |
| `타입 지정 교체` | 특정 타입으로 교체 | "벤치의 번개포켓몬과 교체한다" |
| `데미지 포켓몬 교체` | 데미지 받은 포켓몬과 교체 | "데미지를 받고 있는 포켓몬을 1마리 선택해서 배틀 포켓몬과 교체" |

### 📚 덱/패 조작

| 키워드 | 설명 | 효과 패턴 예시 |
|--------|------|----------------|
| `덱 서치` | 덱에서 특정 카드를 패로 가져옴 | "덱에서 포켓몬/서포트를 랜덤으로 1장 패로 가져온다" |
| `벤치 전개` | 덱에서 포켓몬을 바로 벤치로 내보냄 | "덱에서 기본 포켓몬을 랜덤으로 1장 벤치로 내보낸다" |
| `덱 조작` | 덱 상단을 확인하거나 순서를 바꿈 | "덱을 위에서부터 1장 보고 원래대로 되돌린다" / "원하는 순서대로 바꾸어 덱의 위로 되돌린다" |
| `패 조작` | 패를 덱으로 되돌리거나 교환 | "패를 N장 선택해서 덱으로 되돌린다. 덱을 N장 뽑는다" |
| `덱 파괴` | 상대 덱을 트래쉬함 | "상대의 덱을 위에서부터 1장 트래쉬한다" |
| `서포트 차단` | 상대가 서포트를 사용할 수 없게 함 | "상대는 패에서 서포트를 꺼내서 사용할 수 없다" |
| `진화 차단` | 상대 진화를 막음 | "상대는 패에서 포켓몬을 꺼내서 배틀 포켓몬을 진화시킬 수 없다" |

### 🌱 진화 관련

| 키워드 | 설명 | 효과 패턴 예시 |
|--------|------|----------------|
| `특수 진화` | 턴 제한 없이 즉시 진화 가능 | "첫 번째 차례나 필드에 처음 나왔을 때라도 진화할 수 있다" |
| `진화 서치` | 기술/특성으로 진화 카드를 가져옴 | "진화하는 카드를 덱에서 랜덤으로 1장 올려서 진화시킨다" |
| `에너지 진화` | 에너지 부착 시 자동 진화 | "에너지를 붙일 때마다 진화하는 카드를 올려서 진화시킨다" |
| `스킵 진화` | 1진화를 건너뛰고 2진화 가능 | "1진화를 건너뛰어 진화시킨다" |

### ⏱️ 사용 제약 (충전/반동)

| 키워드 | 설명 | 효과 패턴 예시 |
|--------|------|----------------|
| `기술 사용 불가` | 사용 후 다음 차례에 기술 사용 불가 | "다음 차례에 이 포켓몬은 「○○」를 사용할 수 없다" / "다음 차례에 이 포켓몬은 기술을 사용할 수 없다" |
| `충전 기술` | 이번 차례에 쓰면 다음 차례 데미지 증가 | "다음 차례에 이 포켓몬의 「○○」의 데미지를 +N한다" |
| `차례 종료` | 사용 시 차례가 바로 종료됨 | "사용하면 차례는 끝난다" |

### 🩺 기타 특수 효과

| 키워드 | 설명 | 효과 패턴 예시 |
|--------|------|----------------|
| `서포트 복사` | 상대의 서포트를 복사해 사용 | "상대의 패에서 서포트를 랜덤으로 1장 앞면을 보고 그 효과를 사용한다" |
| `기술 복사` | 벤치 포켓몬의 기술을 복사 | "벤치 포켓몬이 가진 기술을 1개 선택해서 이 기술로써 사용한다" |
| `포켓몬 체크 피해` | 포켓몬 체크마다 상대에게 피해 | "포켓몬 체크 때마다 상대의 배틀 포켓몬에게 N데미지를 준다" |
| `독 피해 강화` | 독으로 인한 피해를 증가 | "상대의 배틀 포켓몬이 받는 독의 데미지를 +10한다" |
| `HP 조건 피해 강화` | 낮은 HP일 때 데미지 상승 | "남은 HP가 50이하라면 기술이 주는 데미지를 +60한다" |
| `데미지 이동` | 받고 있는 데미지를 다른 포켓몬으로 이관 | "선택한 포켓몬이 받고 있는 데미지를 모두 이 포켓몬에게 옮긴다" |

## 현재 기존 키워드와 제안 키워드 매핑

| 기존 (현재 시트) | 제안 (신규) |
|------------------|-------------|
| `독 부여` | → `독 부여` (유지) |
| `화상 부여` | → `화상 부여` (유지) |
| `마비 부여` | → `마비 부여` (유지) |
| `혼란 부여` | → `혼란 부여` (유지) |
| `잠듦 부여` | → `잠듦 부여` (유지) |
| `독 면역` | → `특수상태 면역` 또는 `잠듦 면역` 으로 통합 |

## 우선 입력 권장 순서

효과가 명확한 아래 카드부터 키워드를 채우는 것을 권장합니다.

1. **상태이상 부여** - 독/화상/마비/혼란/잠듦을 부여하는 기술/특성 보유 카드
2. **데미지 감소** - 특성으로 데미지를 -N하는 카드
3. **에너지 부착** - 기술/특성으로 에너지를 즉시 부착하는 카드
4. **상대 강제 교체** - 상대 포켓몬을 강제로 교체시키는 카드
5. **HP 회복** - 회복 효과 보유 카드
6. **덱 서치 / 벤치 전개** - 덱을 조작하는 기술/특성 보유 카드
